## 🛠️ Build & Run a Full Node

**iBitHub** is a Litecoin 0.8-era Scrypt fork.  
**Wallet compatibility requires Berkeley DB 4.8 + OpenSSL 1.0.2u** (newer versions will break it).

### 🚀 Updated & Simplified Instructions (March 2026)

**Full step-by-step guide (Native Ubuntu 18.04 + Docker for modern systems) is now at:**

**[👉 Build & Run Guide → https://www.ibithub.com/build.html](https://www.ibithub.com/build.html)**

- Native build on Ubuntu 18.04 LTS (recommended for simplicity)  
- Containerized deterministic Docker build (perfect for Ubuntu 20.04+/22.04+/24.04+/25.10)  
- VPS recommendations, exact `makefile.unix` edits, LD_LIBRARY_PATH fixes, and more
